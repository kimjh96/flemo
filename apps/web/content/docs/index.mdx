**flemo** is a React router for screen transitions.

Use it when you want a web app that flows screen-by-screen the way a native app does. Push, pop,
the transitions between screens, and swipe-back gestures are all built into **flemo** — the
transitions compile to CSS keyframes. You describe how screens flow in plain React, and flemo
handles the rest.

The goal is simple: stacking and unstacking screens — the way native apps move — should feel
natural in plain React code.
