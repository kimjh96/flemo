## shiflo는 flemo를 어떻게 쓰나요

shiflo는 하이브리드 앱이에요. UI는 웹 앱이고, 그걸 iOS와 Android용으로 감쌌어요. 이 방식의
어려운 점은 _웹 뷰처럼 느껴지지 않게_ 만드는 거예요. 탭하면 새 화면이 밀려 들어오고, 뒤로
가기 제스처가 동작하고, 상세 화면이 목록 위로 슬라이드되는 — 그 감각이요. 바로 그 간극을
flemo가 메워요.

shiflo의 화면 간 이동은 flemo 위에서 돌아가요. push와 pop, 좌측 엣지 스와이프 뒤로 가기,
화면 사이의 전환까지 — 모두 평범한 React 코드로 기술하고, 애니메이션은 CSS 키프레임으로
컴파일돼요. 그 결과, 하나의 웹 코드베이스에서 두 플랫폼 모두 네이티브처럼 읽히는 움직임을
가진 하이브리드 앱이 나와요.

## 왜 중요한가요

이게 flemo가 지향하는 증거예요. 실제 사용자의 손에, 두 스토어에 올라간 진짜 앱 — 플레이그라운드가
아니에요. React 앱이나 하이브리드 셸에서 네이티브 같은 화면 전환이 필요하다면, shiflo가 그게
프로덕션에서 어떤 모습인지 보여줘요.
