**flemo**는 화면 전환을 처리하는 React 라우터예요.

웹앱을 네이티브 앱처럼 화면 단위로 흐르게 만들고 싶을 때 써요. push, pop, 화면 사이의 애니메이션과
제스처를 모두 **flemo**가 처리하기 때문에, 라우터와 모션을 따로 엮지 않아도 돼요.

화면이 쌓이고 풀리는 모바일 앱의 흐름을, React 코드만으로 자연스럽게 다루는 게 목표예요.
