**flemo**는 화면 전환을 처리하는 React 라우터예요.

웹앱을 네이티브 앱처럼 화면 단위로 흐르게 만들고 싶을 때 사용해요. push, pop, 화면 사이의
전환, 스와이프 뒤로가기 제스처까지 모두 **flemo**에 내장돼 있어요. 전환은 CSS 키프레임으로
컴파일돼요. 화면이 흐르는 방식만 React로 기술하면 나머지는 flemo가 처리해요.

화면이 쌓이고 풀리는 네이티브 앱의 움직임을, React 코드만으로 자연스럽게 다루는 게 목표예요.
